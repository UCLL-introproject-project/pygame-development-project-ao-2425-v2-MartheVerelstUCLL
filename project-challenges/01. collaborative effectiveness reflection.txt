Voor dit project (en projecten uit andere vakken) heb ik samengewerkt met mijn verloofde, Jonas. Hij is industrieel ingenieur IT-ICT en werkt als test automation specialist. Aangezien we samenwonen, kon ik hem erg makkelijk vragen stellen.
Ik heb het project volledig zelf geschreven: eerst heb ik de tutorial gevolgd, daarna heb ik nog een knop met de spelregels en een tekstveld waarin je je naam kunt invullen toegevoegd. Jonas heeft mij geholpen wanneer ik vastzat. Ik heb ook met hem gebrainstormd over de functionaliteiten die ik zelf aan het project kon toevoegen.
We hebben voornamelijk via WhatsApp en face to face gecommuniceerd. Bij vragen stuurde ik hem een bericht. Soms kon hij antwoorden via de chat, soms kwam hij naar boven om naar de code te kijken en samen enkele dingen te proberen.
De meeste code begreep ik en ook de extra knop en het tekstveld kon ik grotendeels schrijven zonder hulp. Daardoor heb ik het grootste deel van het project alleen gemaakt. Zonder zijn hulp had ik de oplossing wellicht ook gevonden, maar het zou wel langer geduurd hebben. Omdat ik het vooral zelf heb geschreven en altijd eerst zelf gezocht heb naar de oplossing, begrijp ik de code beter dan als ik delen ervan aan hem zou hebben doorgegeven. Voor dit project is het dus niet erg dat de verdeling van het werk eerder onevenwichtig was.
